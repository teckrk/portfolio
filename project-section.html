Online Bookstore - E-Commerce Website

Overview: Developed a fully functional e-commerce website for an online bookstore, focusing on a seamless user experience and robust backend.

Role: Full-Stack Developer

Technologies: HTML, CSS, JavaScript, React, Node.js, MongoDB

Challenges: Implementing a secure payment gateway and optimizing database queries for faster search results.

Solutions: Integrated Stripe for payment processing and utilized indexing and aggregation in MongoDB to improve search performance by 40%.

Outcome: The website increased the client’s online sales by 25% within the first month of launch. The client praised the intuitive design and speed of the website.
